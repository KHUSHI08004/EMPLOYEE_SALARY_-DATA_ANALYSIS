# EMPLOYEE_SALARY_-DATA_ANALYSIS
💼 Employee Salary Analysis 📊
Python • SQL • Excel • Data Analysis
A complete data analysis project that explores employee salary patterns across departments, experience levels, and other factors using SQL, Excel, and Python.
📌 Overview
This project analyzes an employee dataset containing details such as department, experience, salary, education level, and location.
The goal is to extract meaningful insights that help HR and management understand salary distribution and improve decision-making.
🎯 Objectives
Analyze salary distribution across departments
Calculate average salary in the organization
Study the relationship between experience and salary
Identify highest-paid employees
Generate insights for HR decision-making
🔑 Key Features
📊 Multi-tool workflow — SQL + Excel + Python
🧠 Real-world HR dataset analysis
📈 Visualizations using charts and graphs
🔍 Insight-driven approach (not just raw data)
⚡ Easy-to-understand outputs
📊 Insights
Finance and Marketing departments offer higher salaries
Employees with more experience generally earn more
