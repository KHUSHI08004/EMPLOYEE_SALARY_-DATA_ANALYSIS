# EMPLOYEE_SALARY_-DATA_ANALYSIS
Companies maintain large amounts of employee data including name, department, experience, salary, and location. However, this data is often not analyzed effectively..This analysis helps management make better decisions regarding salary structure and workforce planning.
Tools Used:SQL (SQlite) ,Excel (Pivot Tables, Charts) ,Python (Google Colab,Matplotlib,Pandas)
Key Features:Display and manage employee records using SQL, Calculate average salary ,Identify highest-paid employees, Analyze department-wise salary,BVisualize data using charts and graphs
